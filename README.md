# FastAPI User Authentication and Task Management

Это приложение на FastAPI предоставляет систему аутентификации пользователей с возможностью управления задачами. Пользователи могут регистрироваться, входить в систему, обновлять профиль, удалять аккаунт и управлять своими задачами.

## Функциональность

### Аутентификация пользователей
- Регистрация с именем, фамилией, отчеством, email и паролем
- Вход в систему по email и паролю
- Выход из системы
- Обновление профиля пользователя
- Мягкое удаление аккаунта (is_active=False)

### Управление задачами
- Создание задач
- Просмотр списка задач
- Просмотр конкретной задачи
- Обновление задач
- Удаление задач

## Запуск приложения

### С использованием Docker Compose (рекомендуется)

1. Запустите приложение:
   ```bash
   docker-compose up --build
   ```

2. Приложение будет доступно по адресу: http://localhost:8000

3. Документация API: http://localhost:8000/docs

## API Эндпоинты

### Аутентификация
- `POST /auth/register` - Регистрация пользователя
- `POST /auth/token` - Получение токена доступа
- `POST /auth/logout` - Выход из системы

### Пользователи
- `GET /users/me` - Получение профиля текущего пользователя
- `PUT /users/me` - Обновление профиля
- `DELETE /users/me` - Удаление аккаунта

### Задачи
- `POST /api/tasks` - Создание задачи
- `GET /api/tasks` - Получение списка задач
- `GET /api/tasks/{task_id}` - Получение конкретной задачи
- `PUT /api/tasks/{task_id}` - Обновление задачи
- `DELETE /api/tasks/{task_id}` - Удаление задачи
